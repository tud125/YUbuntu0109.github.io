学习笔记 : Java设计模式之Singleton Pattern

概念

单例设计模式 : 确保一个类只能有一个实例,并提供一个全局访问点来访问这个实例. (Singleton Pattern : Ensure a class has only one intance,and provide a global point of access to it. ). 单例设计模式是一种对象创建型模式,其主要有三个要点 :

  1. 某个类只能有一个实例
  2. 必须自行创建这个实例
  3. 必须自行向系统提供这个实例
